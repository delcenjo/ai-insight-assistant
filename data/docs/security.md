# Security Policy

## Access control

Two-factor authentication is **mandatory** for all company accounts, which are
accessed through single sign-on. Access follows the principle of least privilege
and is reviewed quarterly.

## Passwords

Passwords must be at least **12 characters** and stored in the company password
manager. They must be rotated immediately if a compromise is suspected.

## Data handling

Data is classified as public, internal or confidential. Confidential data must
be encrypted at rest. Suspected security incidents must be reported to the
security team within **one hour** of discovery.
